reformat-type-and-instance-tags

It is possible to reformat how type and instance tags are generated by defining a custom tag definition. This can be set for a project configuration, so all type_tags and instance tags are follow this definition.

Equipment Tags
Projects contain multiple identical (equipment) type_tag's. bdns_tag's and instance_tag's are unique.
